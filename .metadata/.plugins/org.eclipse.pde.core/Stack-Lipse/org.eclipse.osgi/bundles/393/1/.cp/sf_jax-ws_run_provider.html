Running a JAX-WS Service Provider Project

To run your Service Provider:
  1. Click Run > Run Configurations... .
  2. Right-click OSGi Framework and click New. Type a name for your provider configuration in the Name field.
  3. Click the Bundles tab, select the Workspace check box and select the check boxes corresponding to the provider projects you want to run.

    Note: You can use a single configuration to run more than one provider.

  4. Click Apply. Click Run.
