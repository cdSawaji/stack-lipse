Getting Started with Swordfish

This chapter describes how to configure Eclipse for the Swordfish target platform.

To configure Eclipse for the Swordfish platform:
  1. Click Windows > Preferences > Plug-in development > Target Platform and click Add.
  2. In the New Target Definition window, select the Template check box and select Swordfish. Click Next.
  3. Create a new target definition and click Finish.
  4. In the Target Definitions window, select the Swordfish check box. Click Apply.
  5. Click OK to confirm your actions.
