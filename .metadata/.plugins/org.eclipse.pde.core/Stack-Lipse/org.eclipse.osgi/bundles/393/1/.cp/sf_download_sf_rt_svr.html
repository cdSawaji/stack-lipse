Downloading and Installing Swordfish Runtime Server

  1. Start a web browser and go to http://www.eclipse.org/swordfish/downloads/.
  2. Download the zip file containing the Swordfish Runtime Server that is appropriate for your operating system.
  3. Unzip the file and extract it to a chosen directory.
  4. Start the installation wizard and follow the installation instructions.
  5. For instructions on starting, stopping, and configuring the server, refer to the corresponding readme.txt file.
